What is TFMS

Agronomy and factory operations — from a single platform

TFMS was designed from real operational experience managing enterprise agronomy software for Universal Leaf Tobacco across Malawi and Mozambique. It covers every step from initial grower registration to factory intake.

TFMS Registry

Module 01

Digital farmer onboarding with photo biometrics, fingerprint capture, GPS field mapping, household data, and printable grower ID cards.

From $4,500

TFMS Inputs

Module 02

Seed, fertilizer, and chemical issuance with warehouse dispatch, acknowledgement confirmation, and seasonal grower reconciliation.

From $5,500

TFMS Credit

Module 03

Season credit tracking, loan balances, automatic deduction at factory, risk flags, and full credit ledger per grower.

From $6,500

Technology stack

Built for real-world agricultural environments

Offline-first mobile on Android tablets and phones. Windows desktop for back-office and factory capture. Central cloud API for sync and reporting.

C# · .NET 8

Core application logic, shared models, and all business rules — cross-platform via .NET MAUI and ASP.NET Core.

.NET MAUI Android

Field visit app and farmer registry on Android tablets and smartphones — offline-capable, GPS, camera, biometrics.

Windows Forms / WPF

Factory intake, inventory, and back-office administration running on Windows 10/11 tablets and desktops.

SQL Server · SQLite

Central SQL Server for cloud data, SQLite for local offline storage on devices with automatic two-way sync.
