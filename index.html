What is TFMS

Agronomy and factory operations — from a single platform

Designed from real operational experience managing enterprise agronomy software for Universal Leaf Tobacco across Malawi and Mozambique. Every step from grower registration to factory intake.

Farmer-centric from day one
Digital onboarding with photo biometrics, fingerprint capture, GPS field mapping, and printable grower ID cards — every farmer traceable from field to factory.
Offline-first field operations
Works without connectivity in the field. GPS check-in, offline sync, DroidSurvey-compatible forms, and automatic two-way reconciliation when back in range.
Factory-floor precision
Tablet-ready weight and grade capture, bale traceability, curing and processing records, and full inventory movement logging from Shift A to final dispatch.

Technology Stack

Built for real-world
agricultural environments

Offline-first on Android. Windows for factory and back-office. Central cloud API for sync and reporting.

C# · .NET 8
Core application logic, shared models, and all business rules — cross-platform via .NET MAUI and ASP.NET Core.
.NET MAUI Android
Field visit app and farmer registry on Android tablets and smartphones — offline-capable, GPS, camera, biometrics.
Windows Forms / WPF
Factory intake, inventory, and back-office administration running on Windows 10/11 tablets and desktops.
SQL Server · SQLite
Central SQL Server for cloud data, SQLite for offline device storage with automatic two-way sync.
