# TFMS — TechNexus Farmer Management System

A modular agronomy and factory management system for tobacco and agricultural operations.
Built by TechNexus · Lilongwe, Malawi · www.technexusmw.com

## Files

| File | Purpose |
|---|---|
| `index.html` | Main product landing page |
| `about.html` | About the product and developer |
| `modules.html` | Detailed breakdown of all 6 modules |
| `pricing.html` | Module and suite pricing (USD + MWK) |
| `contact.html` | Contact and enquiry form |
| `tfms-application-mockup.html` | Interactive app prototype |
| `style.css` | Shared stylesheet for all pages |

## Stack
- C# · .NET 8 · .NET MAUI (Android)
- Windows Forms / WPF (desktop back-office)
- ASP.NET Core Web API
- SQL Server · SQLite (offline)
- HTML5 / CSS3 (this site)

## Modules
1. TFMS Registry — Farmer onboarding and biometrics
2. TFMS Inputs — Inputs distribution
3. TFMS Credit — Credit management
4. TFMS Field — Field visit monitoring
5. TFMS Factory — Factory processing and inventory
6. TFMS Analytics — Dashboards and reporting

## Deploy
Push all files to `main` branch. Enable GitHub Pages from repository Settings → Pages → Source: main / root.
